## 2. DATOS PERSONALES RECOPILADOS

### 2.1. Datos de Registro e Identificación

Al crear una cuenta, recopilamos:

- **Nombre completo** (nombre y apellido)
- **Correo electrónico** (obligatorio para registro)
- **Número de teléfono** (opcional, pero recomendado para verificación)
- **Contraseña** (encriptada mediante hash, nunca almacenada en texto plano)
- **Fecha de nacimiento** (opcional, para verificación de mayoría de edad)
- **Documento de identidad** (para vendedores y verificación de identidad)

### 2.2. Datos de Perfil

Cuando completa su perfil, podemos recopilar:

- **Foto de perfil** (opcional)
- **Biografía o descripción personal**
- **Ubicación** (ciudad, departamento)
- **Información de contacto adicional**
- **Preferencias de notificación**

### 2.3. Datos de Vendedor/Tienda

Para usuarios que operan como vendedores:

- **Información de la tienda** (nombre, descripción, logo, imágenes)
- **Datos bancarios** (número de cuenta, banco, titular) para pagos
- **Información fiscal** (cuando corresponda)
- **Dirección comercial** (si aplica)

### 2.4. Datos de Transacciones

Al realizar compras o ventas:

- **Historial de compras y ventas**
- **Información de pago** (método utilizado, monto, fecha)
- **Direcciones de envío** (nombre completo, dirección, ciudad, departamento, código postal, teléfono)
- **Información de facturación** (cuando aplica)
- **Comprobantes de pago** (imágenes de transferencias bancarias cuando se envían)

### 2.5. Datos de Comunicación

Cuando utiliza el sistema de mensajería:

- **Mensajes enviados y recibidos** entre usuarios
- **Archivos adjuntos** (imágenes, documentos) compartidos
- **Historial de conversaciones**

### 2.6. Datos de Navegación y Uso

Mediante tecnologías de seguimiento:

- **Dirección IP** y ubicación geográfica aproximada
- **Tipo de dispositivo** (computadora, móvil, tablet)
- **Navegador web** y sistema operativo
- **Páginas visitadas** y tiempo de permanencia
- **Búsquedas realizadas**
- **Productos visualizados**
- **Acciones realizadas** (clics, scroll, interacciones)

### 2.7. Datos de Subastas

Cuando participa en subastas:

- **Historial de pujas realizadas**
- **Monto de pujas**
- **Fecha y hora de cada puja**
- **Estado de participación** (ganador, perdedor, remiso)

### 2.8. Cookies y Tecnologías Similares

Mediante cookies y tecnologías de seguimiento:

## 4. ALMACENAMIENTO Y SEGURIDAD

### 4.1. Proveedor de Almacenamiento

Los datos personales se almacenan en:

**Supabase (Supabase Inc.)**
- Servicios de base de datos y autenticación
- Ubicación de servidores: [ESPECIFICAR REGIÓN]
- Política de privacidad: https://supabase.com/privacy
- Certificaciones: SOC 2 Type II, ISO 27001

### 4.2. Medidas de Seguridad Técnicas

Implementamos las siguientes medidas de seguridad:

- **Encriptación en tránsito:** TLS/SSL para todas las comunicaciones
- **Encriptación en reposo:** Datos sensibles encriptados en la base de datos
- **Autenticación:** Sistema de autenticación seguro mediante Supabase Auth
- **Row Level Security (RLS):** Políticas de seguridad a nivel de fila en la base de datos
- **Contraseñas:** Almacenadas mediante hash bcrypt, nunca en texto plano
- **Tokens de acceso:** Tokens JWT con expiración automática
- **Backups:** Copias de seguridad regulares y encriptadas

### 4.3. Medidas de Seguridad Organizativas

- **Acceso restringido:** Solo personal autorizado accede a datos personales
- **Capacitación:** Personal capacitado en protección de datos
- **Auditorías:** Revisiones periódicas de seguridad
- **Incidentes:** Plan de respuesta a incidentes de seguridad

### 4.4. Retención de Datos

Conservamos sus datos personales:

- **Datos de cuenta:** Mientras su cuenta esté activa y hasta 2 años después de su cierre
- **Datos de transacciones:** 7 años (requisito legal fiscal)
- **Datos de comunicación:** 1 año después del último mensaje
- **Logs de acceso:** 90 días
- **Datos de analítica:** 26 meses (Google Analytics)

Al finalizar los plazos de retención, los datos se eliminan de forma segura o se anonimizan.

## 7. DERECHOS DEL TITULAR DE DATOS

Según la Ley 1682/2001, usted tiene los siguientes derechos:

### 7.1. Derecho de Acceso

Puede solicitar información sobre:

- Qué datos personales tenemos sobre usted
- Para qué finalidad se utilizan
- A quién se comparten
- Por cuánto tiempo se conservan

### 7.2. Derecho de Rectificación

Puede solicitar la corrección de datos:

- Inexactos
- Incompletos
- Desactualizados

### 7.3. Derecho de Supresión

Puede solicitar la eliminación de datos cuando:

- Ya no sean necesarios para las finalidades originales
- Retire su consentimiento (cuando el consentimiento sea la base legal)
- Los datos hayan sido tratados ilícitamente
- Sea necesario para cumplir una obligación legal

**Nota:** Algunos datos pueden conservarse por períodos legales (ej: datos fiscales por 7 años).

### 7.4. Derecho de Oposición

Puede oponerse al tratamiento de datos cuando:

- El tratamiento se base en interés legítimo
- Los datos se utilicen para marketing directo
- Tenga motivos legítimos para oponerse

### 7.5. Derecho de Portabilidad

Puede solicitar recibir sus datos en formato estructurado y de uso común para transferirlos a otro responsable.

### 7.6. Derecho de Revocación del Consentimiento

Puede revocar el consentimiento otorgado en cualquier momento, sin afectar la licitud del tratamiento previo.
